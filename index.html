Ma page HTML

Bienvenue sur ma page HTML simple.

Voici quelques exemples de balises HTML :

  • Balise <h1> : Utilisée pour les titres principaux.
  • Balise <p> : Utilisée pour les paragraphes de texte.
  • Balise <ul> : Utilisée pour créer une liste non ordonnée.
  • Balise <li> : Utilisée pour chaque élément de liste.
